/------MUL_SW Function------
MUL_SW:	HEX 0	/Where return address will be saved when function is called
	STA MULT1	/Store AC as the first value
	LDA FNC_IN	/Load the inputted argument
	STA MULT2	/Store it as the second value
	CLA
	STA PRODUC	/Initialize the output to zero
	LDA CTR_16
	STA MCTR	/Initialize the multiplication counter to -16

M_LOOP:	CLE
	LDA MULT2
	CIR
	STA MULT2	/Shifted the next bit in line from mult2 to E
	SZE		/If our E = 1 Add the weighted mult1 to produc
	BUN M_ADD
	BUN M_NEXT

M_ADD:	LDA PRODUC
	ADD MULT1	/Add our mult1 to produc
	STA PRODUC

M_NEXT:	CLE		/Were preparing for a shift not a cil
	LDA MULT1
	CIL		/Weigh MULT1 more for the next pass
	STA MULT1
	ISZ MCTR
	BUN M_LOOP

	LDA PRODUC	/When the multiplication is done, load the product into AC
	BUN MUL_SW I	/Return




/------DIV_SW Function------
DIV_SW:	HEX 0		/Return address
	STA DIV1	/Store AC as the Dividend
	LDA FNC_IN	/FNC_IN is our divisor
	CMA
	INC
	STA DIV2	/Perform 2s compliment to obtain negative divisor
	CLA
	STA QUOT	/Initialize quotient to 0

D_LOOP:	LDA DIV1
	ADD DIV2	/Subtract divisor from dividend
	SPA		/Check if were still positive
	BUN D_DONE	/If were not, then the division is over
	STA DIV1	/Update dividend
	ISZ QUOT	/Increment quotient
	BUN D_LOOP

D_DONE:	LDA QUOT	/Load the quotient to AC when done
	BUN DIV_SW I	/Return




/------FACT Function------
FACT:	HEX 0		/Return Address
	STA FVALUE	/Save the value of AC to FVALUE
	CMA
	INC
	STA FCTR	/Use a negative version of FVALUE as our counter
	LDA ONE
	STA FFINAL	/Initialize the final value to 1

F_LOOP:	LDA FVALUE
	STA FNC_IN	/Load FNC_IN with FVALUE
	LDA FFINAL	/Load AC with FFINAL
	BSA MUL_SW	/Multiply FFINAL and FVALUE
	STA FFINAL	/Update FFINAL with the result
	LDA FVALUE
	ADD NEGONE	/Decrement FVALUE
	STA FVALUE
	ISZ FCTR
	BUN F_LOOP
	
	LDA FFINAL	/Load the final result of the factorial
	BUN FACT I	/Return




/------FIBB Function------
FIBB:	HEX 0		/Return Address

	INC
	INC		/Increment twice cause the function starts one later than usual
	STA FIBCTR	/The value in AC will be used to count the iterations through the sequence
	LDA ONE
	STA FIBONE	/Initialize FIBONE with 1
	LDA ZERO

FIBB_L:	LDA FIBONE

	STA FIBVAL
	LDA FIBONE

	LDA FIBVAL
	STA FIBONE	/Update F(n-1) to be F(n)
	ISZ FIBCTR

	BUN FIBB I	/Return

/------Linked List Traverse------
TRVRSE:	HEX 0
	CMA
	INC		/Store the value of the Xth element in the list to traverse to 
	STA TRCTR	/as a counter (negative)
	LDA LLHEAD	/Load the head of the LL and store it in our pointer (LLNEXT)
	STA LLNEXT

	ISZ TRCTR	/If were at the index we want return
	BUN TLOGIC
	BUN TRDONE

	BUN TRDONE	/Return a 0 if index out of bounds

	BUN TRLOOP

	SZA

	INC

	BUN LLDELT I	/Return

NOHEAD:	CMA

	STA DLCTR	/as a counter (negative)
	LDA LLHEAD	/Load the head of the LL and store it in our pointer (LLNEXT)
	STA LLNEXT	
	
LOOPDL:	ISZ DLCTR	/If were at the index we want start deleting
	BUN NODLTE

	LDA LLNEXT

	SZA		/Check if what were deleting is null

FINALD:	STA DLPTR	/Now were pointing to the node to be deleted

	ISZ LLNEXT

NODLTE:	ISZ LLNEXT	/If not look at the nodes pointer

	SZA		/If pointing to null just return

	BUN LOOPDL

DLDONE:	BUN LLDELT I

/------Section 4.6------
DIV_FP: HEX 0           / Return address.
        STA DF_VAL      / Store the original dividend passed in AC.
        LDA SCALE_N     / Load the negative scaling factor (e.g., -8).
        STA DF_CTR      / Set up shift counter.
S_UP:   LDA DF_VAL      / Load dividend.
        CLE             / Clear E bit to ensure a 0 is shifted in.
        CIL             / Circular Left Shift (effectively AC = AC * 2).
        STA DF_VAL      / Save doubled value.
        ISZ DF_CTR      / Repeat N times.
        BUN S_UP
        LDA DF_VAL      / Load the now-scaled dividend.
        BSA DIV_SW      / Perform standard integer division.
        STA DF_RES      / Store the result.
        LDA DIV1        / Load the remainder left over by DIV_SW.
        CLE             / Prepare to multiply remainder by 2.
        CIL             / AC = Remainder * 2.
        CMA             / Negate (Rem*2).
        INC
        ADD FNC_IN      / Compare: Divisor - (Remainder * 2).
        SPA             / If result is positive, Rem*2 < Divisor (No rounding).
        BUN DF_DONE     / Finish if no rounding needed.
        LDA DF_RES      / Otherwise, round up.
        INC             / Increment the quotient.
        STA DF_RES
DF_DONE:LDA DF_RES      / Load final result to AC.
        BUN DIV_FP I    / Return.

/--- Parameters and Variables ---
SCALE_N: DEC -8         / Scaling factor parameter (2^8 = 256)
DF_VAL:  HEX 0          / Temporary dividend
DF_CTR:  HEX 0          / Shift counter
DF_RES:  HEX 0          / Intermediate result
DIV_REM: HEX 0          / Remainder (must be saved by your DIV_SW)

/------Closest Element Function------
CLOSEST: HEX 0           / Return Address.
         LDA MAX_VAL     / Initialize min_diff with a very large number (7FFF).
         STA MIN_DIFF
         CLA
         STA BEST_IDX    / Store index of the best match found so far.
         STA CUR_IDX     / Current index we are inspecting.
         LDA L_SIZE      / Setup negative loop counter.
         CMA
         INC
         STA L_CTR
C_LOOP:  LDA L_PTR I     / Load array element A[i].
         CMA
         INC             / -A[i].
         ADD FNC_IN      / AC = TargetValue - A[i].
         SPA             / If result is negative, convert to absolute value.
         BUN C_ABS
         BUN C_CHECK
C_ABS:   CMA             / Two's complement for absolute difference.
         INC
C_CHECK: STA AC_TMP      / This is current distance |Target - A[i]|.
         CMA
         INC             / -(current distance).
         ADD MIN_DIFF    / Compare: MinDiff - CurrentDistance.
         SPA             / If positive, MinDiff > CurrentDistance (New closer match!).
         BUN C_NEXT      / If negative, skip; current element is further away.
         LDA AC_TMP      / Load new smaller distance.
         STA MIN_DIFF    / Update minimum distance record.
         LDA CUR_IDX     / Load current position.
         STA BEST_IDX    / Update best index record.
C_NEXT:  ISZ L_PTR       / Point to next element.
         LDA CUR_IDX
         INC
         STA CUR_IDX     / Increment index tracker.
         ISZ L_CTR       / Loop control.
         BUN C_LOOP
         LDA BEST_IDX    / Load the index of the closest element into AC.
         BUN CLOSEST I   / Return.


/------Word Search Function------
W_SEARCH: HEX 0         / Return address.
        CLA             / Clear AC.
        STA AC_TMP      / This will be our match counter.
        STA CURR_IDX    / Tracks which index (0, 1, 2...) we are currently at.
        LDA L_SIZE      / Standard negative loop counter setup.
        CMA
        INC
        STA L_CTR
W_LOOP: LDA L_PTR I     / Load word from current sentence position.
        CMA             / Compare with target word in FNC_IN.
        INC
        ADD FNC_IN      / AC = Target - Word.
        SZA             / If AC is not zero, they are not equal.
        BUN W_NEXT      / Skip if no match.
        LDA CURR_IDX    / Match found! Load the current index number.
        STA L_PTR2 I    / Save this index into the results memory array.
        ISZ L_PTR2      / Move the result pointer forward for the next match.
        ISZ AC_TMP      / Increment the total match count.
W_NEXT: ISZ L_PTR       / Move sentence pointer to next word.
        ISZ CURR_IDX    / Increment index counter.
        ISZ L_CTR       / Loop check.
        BUN W_LOOP
        LDA AC_TMP      / Load total count into AC.
        BUN W_SEARCH I  / Return.


/------Bubble Sort Function------
B_SORT: HEX 0           / Return address.
        LDA L_SIZE      / Setup outer loop counter.
        CMA
        INC
        STA O_CTR       / Outer counter (Passes through the array).
O_LOOP: LDA L_PTR       / Reset the inner pointer to the start of the array.
        STA I_PTR
        LDA L_SIZE      / Setup inner loop counter.
        CMA
        INC
        STA I_CTR
I_LOOP: LDA I_PTR I     / Load current element (A).
        STA VAL_A       / Save A.
        ISZ I_PTR       / Point to next element (B).
        LDA I_PTR I     / Load element B.
        STA VAL_B       / Save B.
        CMA             / Prepare to subtract: A - B.
        INC
        ADD VAL_A       / If A > B, AC will be positive.
        SPA             / If AC is positive, A > B, so we must SWAP.
        BUN SWAP
        BUN I_NEXT      / Otherwise, they are in order, skip swap.
SWAP:   LDA I_PTR       / Backtrack to find address of Element A.
        ADD NEGONE      / I_PTR - 1.
        STA ADR_TMP     / Save A's address.
        LDA VAL_B       / Take B's value.
        STA ADR_TMP I   / Write it into A's original memory slot.
        LDA VAL_A       / Take A's value.
        STA I_PTR I     / Write it into B's memory slot.
I_NEXT: ISZ I_CTR       / Finish inner pass.
        BUN I_LOOP
        ISZ O_CTR       / Finish outer pass.
        BUN O_LOOP
        BUN B_SORT I    / Array is now sorted!
