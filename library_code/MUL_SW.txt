Below is the source code for the MUL_SW function, it is included in the 'A_FULLLIBRARY.txt' file. Copy that at the end of your program to include the library

Function: Multiplies two numbers together, the value in AC and the value stored at FNC_IN.
	  Output is returned in AC
Code:
MUL_SW:	HEX 0	/Where return address will be saved when function is called
	STA MULT1	/Store AC as the first value
	LDA FNC_IN	/Load the inputted argument
	STA MULT2	/Store it as the second value
	CLA
	STA PRODUC	/Initialize the output to zero
	LDA CTR_16
	STA MCTR	/Initialize the multiplication counter to -16

MLOOP:	CLE
	LDA MULT2
	CIR
	STA MULT2	/Shifted the next bit in line from mult2 to E
	SZE		/If our E = 1 Add the weighted mult1 to produc
	BUN M_ADD
	BUN M_NEXT

M_ADD:	LDA PRODUC
	ADD MULT1	/Add our mult1 to produc
	STA PRODUC

M_NEXT:	CLE		/Were preparing for a shift not a cil
	LDA MULT1
	CIL		/Weigh MULT1 more for the next pass
	STA MULT1
	ISZ MCTR
	BUN MLOOP

	LDA PRODUC	/When the multiplication is done, load the product into AC
	BUN MUL_SW I	/Return


/------Local Variables------
MCTR:	DEC 0	/Generic Counter Variable
PRODUC:	HEX 0	/The returned value for the MUL_SW function to be loaded to AC
MULT1:	HEX 0	/First Value to be multiplied (AC)
MULT2:	HEX 0	/Second Value to be multiplied (Input from FNC_IN)



/------Constants------
CTR_16:	DEC -16



/------Inputs------
FNC_IN:	HEX 0	/The input to the functions used by the library
